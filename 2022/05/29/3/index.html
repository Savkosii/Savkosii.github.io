What If Duplicate Icon Pops Out In the Bar When Launching the Program?

Do the following steps:

  1. Press Alt + F2

  2. Input command lg

  3. Click on the Windows tab.

  4. You will see the titles of all your windows there. Below each one you will find a wmclass property with a specific value. Copy the corresponding value of wmclass owned by your program window.

  5. Open the .desktop file, add a new line StartupWMClass= , and paste the value you have copied. After that, the .desktop file may seem like:

  6. Save that file. When you relaunch the program, you will find there is no duplicate icon anymore :)
