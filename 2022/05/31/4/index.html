Case 1 : The Repository is fairly New and Empty

Then, add files in that folder and commit changes. Note that you can add all files in folder with flag -A

Finally, add new remote origin and push to Github.

The -u flag sets the remote origin as the default. This lets you later easily just do git push and git pull without having to specifying an origin.
