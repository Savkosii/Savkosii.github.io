Cat.c

Maximal Square

题目描述

给定一个二维的 0-1 矩阵，求全由 1 构成的最大正方形的面积

输入输出样例

输入是一个二维数组，输出一个 32 位整型

题解

设 dp[i][j] 表示以 (i,j) 所在的 entry 为右下角的最大正方形的长度，比如对于矩阵 , 以 (0,1) 为右下角的最大正方形即它本身，长为 1 ，所以 dp[0][1] = 1；以 (0, 3) 为右下角没有正方形，因为 (0, 3) 对应的元素为 0 ，所以 dp[0][3] = 0 ；以 (1, 1) 为右下角的正方形最大边长为 2 ，所以 dp[1][1] = 2

当 (i,j) 所在的 entry 为 1 时，以之为右下角的最大正方形的长度取决于其分别以左、右、左上 entry 为右下角的最大正方形的长度中的最小者（可以用假设法分析，如果不选取最小者，则补入 (i, j) 这个 entry 后构成不了一个新的正方形），换言之，

因此，初始化时将 dp 置 0，而且遍历时还是需要讨论边界。第 0 行和第 0 列所在 entry 只可能构成边长为 0 和边长为 1 的正方形，且构成情况恰取决于自身是 0 还是 1
